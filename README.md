# FNP Sales Analysis Dashboard

## Project Overview

This project analyzes sales performance for **Flowers & Petals (FNP)** using an Excel dashboard to uncover insights related to customer purchasing behavior, product demand, seasonal trends, and revenue distribution.

The dashboard provides an interactive overview of sales metrics, enabling data-driven decision making for marketing strategies, product optimization, and regional demand analysis.

---

## Objectives

The main objectives of this project are:

- Analyze overall sales performance
- Identify high-performing product categories
- Understand occasion-based demand patterns
- Track monthly revenue trends
- Evaluate customer spending behavior
- Analyze city-wise order distribution

## Key Insights

### Occasion-Based Revenue
Anniversary and Raksha Bandhan generate the highest revenue, showing strong demand during gifting occasions.

### Product Category Performance
Top-performing categories include:

- Colors
- Soft Toys
- Sweets

### Monthly Sales Trend
Sales spikes occur during:

- February (Valentine's Day)
- August (Raksha Bandhan)
- November (Festive season)

### Product Performance
Top revenue generating products:

- Magnam Set
- Quia Gift
- Dolores Gift
- Harum Pack

### Geographic Distribution
Cities such as **Imphal, Kavali, and Dibrugarh** show the highest order volume.
